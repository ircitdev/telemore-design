🚀 Что появилось в Вехе 3

Со времён m2 закрыли весь блок A1–A11 — Админка продюсера из дизайна, плюс выложили три неоновых эффекта в стиле Vice City. На главной для PRODUCER/SUPERADMIN теперь полноценный дашборд вместо лаунчера.

  • ★
    📊 A1 — Dashboard продюсера — главный экран со сводкой смены. 4 KPI-плитки с дельтами: УЧАСТНИКИ (новые сегодня), СРЕДНИЙ БАЛАНС (сумма всех ТК), ЧЕК-ИНОВ (% явка), СОБЫТИЯ (идут сейчас / завершено). Битва отрядов компактным блоком, 6 action-кнопок (Сканер · Bulk · История · Рассылка · События · Участники), события сегодня с LIVE-бейджем, топ-5 + ачивки, лента последних 5 начислений всех продюсеров. Авто-обновление каждые 30 сек.
  • ★
    👥 A3 — Список участников — все 60 одним экраном. Live-поиск по имени/фамилии/@username (debounced на клиенте), фильтр-чипы по отрядам с подсчётами, сортировка Имя / Баланс / Активность (последняя транзакция). На карточке: аватар-инициалы, отряд, @username, число ачивок, баланс. Тап → A4.
  • ★
    🪪 A4 — Карточка участника — полный профиль с аватаром, балансом (rolling Counter), ачивками. Quick-actions: + НАЧИСЛИТЬ (ведёт на /award с QR) и ✉ СООБЩЕНИЕ (one-to-one рассылка). Раскрывающаяся «Анкета» — продюсер меняет имя/фамилию/телефон с маской/ДР/отряд через PATCH; изменения логируются, участник получает DM с diff, при смене отряда — рассылка новому отряду. Полная история начислений (50 последних) с цветовой кодировкой и зачёркнутыми откатами.
  • ★
    📅 A5/A6 — Редактор события + ручной чек-ин — на странице события появились две кнопки. ✏ РЕДАКТ. открывает Sheet с теми же полями что и при создании (PATCH). + ЧЕК-ИН — список всех участников тура с поиском, тап → manual чек-ин без QR (для случаев когда у участника разрядился телефон). У каждой строки чек-ина теперь × — удаление с confirm, при наличии auto-XP создаётся reverse-транзакция и баланс компенсируется.
  • ★
    🪪 A11 — Генератор бейджей — все 60 участников превращаются в печатные бейджи 100×150 мм одним нажатием. Layout «Vice City Full» из badges-preview: фон-плакат с закатом и пальмами, TELEMORE-wordmark с pink→cyan градиентом, имя+отряд, QR. Кнопка 🖨 ПЕЧАТЬ / PDF через tg.openLink открывает страницу в системном браузере (Telegram блокирует window.print()), JWT передаётся через ?token=, авто-печать после загрузки. Каждый бейдж на отдельной странице A6 portrait, фильтр по отрядам.
  • ★
    ✨ Эффекты в стиле Vice City — SpotlightCard: радиальный «прожектор» в точке курсора/тапа на KPI и action-плитках (pink/cyan/gold по тону карточки). AnimatedFeed: staggered scale-in 0.85→1 для лент «События сегодня» и «Последние начисления» + лёгкие edge-fade. Counter: rolling-odometer для баланса участника — после начисления цифра катится с старого значения на новое прямо в карточке (на /producer/award и /producer/participants/[id]).
  • ★
    🔔 Уведомления и обновлённый бот — после каждого начисления участник получает личное сообщение от бота: «🪙 Тебе начислили телекоины! +25 ТК — Лучшая работа дня · от: Александр Успешный · 💰 баланс: 175 ТК · 🏆 открыта ачивка: Карнавал». Бот переименован: ТЕЛЕМОРЕ — Миссия Сочи, новые description/short в Bot API.

  • ●

  • ●
    Главный экран — баланс Телекоинов крупно, отряд, имя.
  • ●

  • ●
    Анкета — все данные участника одним экраном.
  • ●
    Рейтинг — топ-10, твоё место (если ниже), битва отрядов «Кобры» vs «Акулы». Сортировка: баланс DESC → свежесть транзакции DESC. Авто-обновление 30 сек.

    Личное Telegram-уведомление при начислении: кто, за что, сколько, текущий баланс, новая ачивка.

Для продюсера / суперадмина

  • ●

  • ●

▶ Как тестировать

Тест в две роли: участник (получает Телекоины) и продюсер (начисляет). Можно тестировать одному с двух устройств / разных аккаунтов.

1 — Участник

  1. Открой бота
    В Telegram найди @telemore_bot → жми Старт → жми синюю кнопку «Открыть миссию».
  2. Зарегистрируйся
    Имя / Фамилия / Телефон (вводи цифры — маска применится сама) / Дата рождения / выбери отряд: Кобры или Акулы.
  3. Изучи главный экран
    Баланс «0 Телекоинов», кнопка «Мой QR» — покажи продюсеру для начисления.
  4. Получи Телекоины
    Когда продюсер начислит — придёт уведомление в Telegram с разбивкой: сумма, причина, новый баланс, ачивка (если открылась).
  5. Проверь рейтинг и ачивки
    Кнопки Рейтинг и Ачивки внизу. Ачивки разблокируются автоматически при пересечении порога (100, 200, 300, 400, 500, 600 ТК).

2 — Продюсер: новая админка

Чтобы стать продюсером, твой Telegram-ID должен быть прописан в БД с ролью PRODUCER или SUPERADMIN. Скинь свой @username — добавим. Сейчас доступ есть у заказчика и тимлида.
  1. Зайди — попадёшь сразу на дашборд
    PRODUCER/SUPERADMIN при входе летит сразу в 📊 ПРОДЮСЕР. KPI-плитки тапаются — у каждой подсветка-«прожектор» в точке тапа. Action-плитки тоже.
  2. A3 — Список участников
    Тапни 👥 УЧАСТНИКИ. Введи в поиск пару букв имени или @username — список фильтруется. Поменяй сортировку: Имя / Баланс / Активность. Чипом «Кобры» или «Акулы» отфильтруй по отряду.
  3. A4 — Карточка участника
    Тапни на любую карточку. Раскрой Анкета → поменяй отряд (например «Акулы» → «Кобры») → Сохранить. Участнику придёт DM с diff. Вернись на список — увидишь, что отряд обновился. Тут же можно начислить через + НАЧИСЛИТЬ.
  4. A5/A6 — События
    Зайди в 📅 СОБЫТИЯ → создай тестовое (например «Завтрак, 8:30, +15 ТК»). На странице события: ✏ РЕДАКТ. — меняем поля; + ЧЕК-ИН — отметить участника без QR; на каждом чекине × — снять отметку (с откатом auto-XP, если был).
  5. A11 — Бейджи
    Жми 🪪 ГЕНЕРАТОР БЕЙДЖЕЙ → 60 карточек в стиле Vice City. Жми 🖨 ПЕЧАТЬ / PDF — Telegram откроет страницу в системном браузере → диалог принтера / «Сохранить как PDF». Каждый участник на отдельной странице A6.
  6. Начисление с rolling-counter
    Отскань QR → начисли любую сумму. На карточке участника баланс плавно перекатится с старого значения на новое (Counter-эффект). После 2 сек — обратно на сканер.
Что-то не работает? На главной если упало — приложение покажет ошибку с диагностикой. Скинь скриншот в чат — пофикшу за итерацию (~5 мин).
