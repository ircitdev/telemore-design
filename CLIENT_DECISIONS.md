## 2. Рекомендация по VPS

Для production-ready запуска (60 пользователей, 13 дней) минимум:

| Ресурс | Минимум | Рекомендую |
|--------|---------|-----------|
| vCPU | 2 | **2-4** |
| RAM | 4 GB | **4-8 GB** |
| SSD | 40 GB | **60-80 GB** (на бэкапы и логи) |
| Network | 100 Mbps | 1 Gbps |
| OS | Ubuntu 24.04 LTS | как сейчас |

**Что будет на VPS** (продакшн):
- PostgreSQL 16 (БД)
- Node.js + Fastify (API + бот)
- Redis (опционально, для rate limiting / очередей)
- Nginx (reverse proxy)
- Frontend (Next.js, статика отдаётся nginx)
- Cloudflare Tunnel (SSL без открытия портов)
- Бэкапы pg_dump через cron → R2/S3

**Текущий сервер** (`2.27.4.90`): Ubuntu 24.04, на нём уже работает OpenPencil (дизайн-инструмент) + landing app.telemore.org. Проверить характеристики:

```bash
ssh root@2.27.4.90 'nproc && free -h && df -h /'
```

Если меньше 4GB RAM или 40GB диска — нужно проапгрейдить тариф у хостера до запуска бэкенда.

## 4. Уточнения для БД/API (новые требования)

### Новые поля в таблицах

```sql
-- transactions: возможность отката
ALTER TABLE transactions ADD COLUMN reversed_at TIMESTAMPTZ;
ALTER TABLE transactions ADD COLUMN reversed_by UUID REFERENCES users(id);
ALTER TABLE transactions ADD COLUMN original_transaction_id UUID REFERENCES transactions(id);

-- tours: chat_id для поздравлений
ALTER TABLE tours ADD COLUMN birthday_chat_id BIGINT;
ALTER TABLE tours ADD COLUMN timezone VARCHAR(50) DEFAULT 'Europe/Moscow';

-- tour_participants: история смены отряда
CREATE TABLE squad_changes (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  participant_id UUID REFERENCES tour_participants(id),
  from_squad VARCHAR(100),
  to_squad VARCHAR(100),
  changed_by UUID REFERENCES users(id),
  reason TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- events: возможность авто-начисления
ALTER TABLE events ADD COLUMN auto_xp_on_checkin INT DEFAULT 0;
ALTER TABLE events ADD COLUMN bulk_mode BOOLEAN DEFAULT false;
```

### Новые API endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| `POST` | `/api/transactions/:id/reverse` | Откат транзакции (с указанием причины) |
| `PATCH` | `/api/participants/:id/squad` | Сменить отряд участника |
| `POST` | `/api/scanner/bulk-session/start` | Начать bulk-режим: фиксированная сумма + событие |
| `POST` | `/api/scanner/bulk-session/scan` | Скан в bulk-режиме (одинаковая сумма всем) |
| `POST` | `/api/scanner/bulk-session/finish` | Завершить bulk-сессию + вернуть статистику |

## 6. Новые вопросы клиенту

После твоих уточнений возникли ещё несколько мелких:

### Откат транзакций

1. **Срок отката** — без ограничений (можно откатить даже месяц спустя), или только в течение 24/72 часов?  
   → По умолчанию делаю **без ограничений** (любой продюсер может откатить любую транзакцию с указанием причины). Если нужно ограничить — скажи.
2. **Кто может откатывать?** Только тот продюсер, кто начислил — или любой продюсер? Или только суперадмин?  
   → По умолчанию **любой продюсер** (часто продюсер не заметил ошибку коллеги, нужно быстро откатить).

### Bulk-сканирование

3. **Сценарий**: продюсер открывает «Зарядку», говорит «всем +15», сканирует подряд → каждый получает 15. Если кто-то отсканирован дважды — игнорируем второй скан или начисляем повторно?  
   → По умолчанию **дубль игнорируется** (показ «уже отмечен» с временем).
4. **Завершение сессии**: автоматически через 30 минут бездействия + ручная кнопка «Завершить»?

### Смена отряда

5. **Что происходит с балансом** при смене отряда? Баланс **остаётся у участника** (он унёс свои Телекоины) — это влияет на командный зачёт. ОК?
6. **Кто может менять**: продюсер / суперадмин / сам участник?  
   → По умолчанию **продюсер и суперадмин** (участник не может менять сам, чтобы не было мух между отрядами).

### PDF бейдж для продюсера

7. **Формат печати**: A6 (10×15) одна страница на участника, или A4 с 4-9 бейджами на странице?
8. **Что на бейдже**: только QR крупно, или + имя + отряд для глаза человека?

### Лого TELEMORE

9. У тебя есть **референсы / mood**, какие лого нравятся? GTA Vice City — это один стиль, но варианты в нём: chrome-неон, retrowave, графитти. Скинуть несколько на выбор?

### Конкретный список ачивок (для базовой смены)

10. Я закладываю 6 ачивок шагом 100 XP с такими названиями: Новичок / Активист / Профи / Мастер / Легенда / Босс. Подходят? Или хочешь свои?  
    Через супер-админку всё равно можно будет менять, но первые 6 нужны для запуска.
